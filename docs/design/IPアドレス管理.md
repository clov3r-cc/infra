## 1. 説明

使用しているIPアドレスを管理するドキュメントです。

## 2. セグメント

VLAN ID と CIDR ブロックの第3オクテットが一致するようにしています。

| VLAN ID |        CIDR        |        説明         | DHCP  |
| :-----: | :----------------: | ------------------- | :---: |
|    1    |  `192.168.1.0/24`  | NW機器              |   y   |
|   10    | `192.168.10.0/24`  | 私用                |   y   |
|   20    | `192.168.20.0/29`  | サーバ・管理 NW     |   -   |
|    -    | `192.168.20.17/28` | サーバ、DMZ NW      |   -   |
|    -    | `192.168.21.0/24`  | サーバ・サービス NW |   -   |
|   22    | `192.168.22.0/24`  | サーバ・内部 NW     |   -   |
|   30    | `192.168.30.0/24`  | 仕事用              |   y   |
|   40    | `192.168.40.0/24`  | IoTデバイス         |   y   |

### 2.1. `192.168.1.0/24`（NW機器）

NW機器を接続するセグメントです。

|  第4オクテット  |     説明      |
| :-------------: | ------------- |
|      `.1`       | prd-cor-01    |
|      `.2`       | prd-acc-01    |
|      `.3`       | prd-wap-01    |
|      `.4`       | prd-acc-02    |
| `.200` - `.254` | DHCP 配布対象 |

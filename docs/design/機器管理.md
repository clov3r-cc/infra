## 1. 説明

使用している物理機器及び仮想マシンを管理するドキュメントです。

## 2. 命名規則

機器のホスト名は以下の規則に従って命名します。

`<環境>`-`<役割>`-`<連番（2桁、0埋めあり）>`

### 2.1. 環境

3文字の英数字で表します。

| 識別子 | 意味 |
| :----: | ---- |
|  prd   | 本番 |
|  dev   | 開発 |

### 2.2. 役割

3文字の英数字で表します。

| 識別子 |              意味               |
| :----: | ------------------------------- |
|  cor   | ルータ                          |
|  acc   | L2 スイッチ                     |
|  wap   | 無線 AP                         |
|  pve   | Proxmox ホスト                  |
|  vyo   | VyOS ルータ                     |
|  tal   | Tailscale                       |
|  dsq   | DNS/Proxy サーバ                |
|  zbx   | Zabbix サーバ                   |
|  kcp   | Kubernetes Control Plane ノード |
|  kwk   | Kubernetes Worker ノード        |
|  sen   | センサー機器                    |
|  nas   | NAS（専用機器）                 |

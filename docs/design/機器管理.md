## 1. 説明

使用している物理機器及び仮想マシンを管理するドキュメントです。

## 2. 命名規則

機器のホスト名は以下の規則に従って命名する。

<環境>-<役割>-<連番>

### 2.1. 環境

| 識別子 | 意味 |
| :----: | ---- |
|  prd   | 本番 |
|  tst   | 検証 |

### 2.2. 役割

| 識別子 |              意味               |
| :----: | ------------------------------- |
|  rtr   | ルータ                          |
|  l2s   | L2 スイッチ                     |
|  wap   | 無線 AP                         |
|  pve   | Proxmox ホスト                  |
|  tsc   | Tailscale                       |
|  zbx   | Zabbix Server                   |
|  ans   | Ansible 実行環境                |
|  kcp   | Kubernetes Control Plane ノード |
|  kwk   | Kubernetes Worker ノード        |
|  hsc   | センサー機器                    |
